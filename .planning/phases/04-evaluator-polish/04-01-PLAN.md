# Phase 4: Evaluator Polish & Demo Experience

## Goal
Add evaluator-facing polish: one-click demo room, sound/toast notifications, CSV export, keyboard shortcuts, and spectator count — making the live auction feel production-ready in a single session.

## Mode
mvp

## Depends on
Phase 3

## Requirements
- DEMO-01: One-click demo room creation from landing page
- DEMO-02: Sound notifications for bid placed, timer < 5s, SOLD/UNSOLD resolution
- DEMO-03: Toast notifications replacing alert() for kick/disband/error
- DEMO-04: CSV export button on results page
- DEMO-05: Keyboard shortcuts (Space to bid, P to pause/resume)
- DEMO-06: Spectator count badge in auction header

## Success Criteria
1. Landing page "Try Demo" button creates a pre-populated room and redirects to lobby in < 3 seconds
2. All audio cues play correctly across Chrome/Firefox/Safari (user interaction required first)
3. Toast notifications appear for all former alert() cases (kick, disband, bid rejected, errors)
4. Results page CSV download includes: team, player, category, sold price, purse remaining
5. Space bar places bid when Team Owner has focus and can bid; P toggles pause/resume for Commissioner
6. Header shows "👁 N watching" with live spectator count
